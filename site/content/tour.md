# See it in action

Lesstruct is **one site with one default theme**, and that theme renders every
content type you define in `config.toml` — blog posts, static pages, and your own
custom post types. These shots are captured from a running demo (regenerated by
`make screenshots`) and served in both light and dark themes.

> These are real renders from a single Lesstruct instance, not separate sites.
> The same default theme adapts its layout to each post type.
